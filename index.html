Переменные

Переменная – это «именованное хранилище» для данных. Мы можем использовать переменные для хранения  чисел, товаров, посетителей и других данных

Переменные как коробки для данных

Для создания переменной в JavaScript используется ключевое слово let

Создание переменной

            let message;
            message = 'Привет!';
            alert(message);
  • Можно объявлять несколько переменных в одной строке
  • Для изменения значения переменной не нужно ее заного объявлять

Неизменяемые переменные – константы

            const MESSAGE = 'Привет!';
            MESSAGE = 'Здравствуй!';
            alert(MESSAGE);

Принято записывать имена констант большими буквами

Важно давать понятные имена переменным
